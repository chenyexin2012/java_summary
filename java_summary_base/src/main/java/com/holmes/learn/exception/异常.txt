一、error
	程序无法处理的错误，如OutOfMemoryError
	
二、exception
	程序可以处理的异常，分为非运行时异常（发生在编译阶段，checkException)
	和运行时异常（发生在程序运行过程中，uncheckException）
	
	1.非运行时异常
		一些不遵守java规范的异常，可以直接看出来的异常。
	
	2.运行时异常
		在程序运行时出现的异常，通常具有不确定性，如空指针异常NullPointerException。
	
	
	
三、分类
	1.受控异常与非受控异常
		受控异常通常值非运行时异常
		非受控异常通常指运行时异常和Error
	2.非运行时异常和运行时异常
